## Setup

- Machine: `thinkpad t440p`
- Hostname: `samurai`
- Operating System: `arch`
- Window Manager: `dwm`
- Shell: `zsh`
- Terminal Emulator: `st`
- File Manager: `dmenufm, pcmanfm`
- Application Launcher: `dmenu`
- Compositor: `picom`
- Status Bar: `dwmblocks`
- Notification: `dunst`
- Text Editor: `nvim, nano`
